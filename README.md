This is a Java-based Banking System built as part of a project. The system allows users to create bank accounts, deposit money, withdraw money and transfer money between accounts. The system is split into three parts — a basic banking system, CSV file reading and a graphical user interface.<br>

How to Run the Project:<br>
Open Eclipse IDE<br>
Create a new Java project called BankingSystem<br>
Add all the provided .java files into the BankingSystem package<br>
Place the Accounts.csv file in the root folder of the project (same level as src)<br>
Run the Main.java file to start the program<br>
The GUI window will open automatically<br>


Files Included
FileDescriptionCustomer.java - Stores customer first name and last name<br>
Account.java - Extends Customer, handles balance and account number<br>
Transaction.java - Handles money transfer between two accounts<br>
ReadAccounts.java - Reads account data from the CSV file<br>
GUI.java - Graphical user interface built using Java Swing<br>
Main.java - Entry point — loads accounts and launches the GUI<br>
Accounts.csv - Contains account data (first name, last name, account number, balance)
